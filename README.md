## Collection of utils for Unix-like systems.

### 1. pass-check-health

Util to run some tests against your ~/password-store folder for passwordstore util pass.

Example:
```
$ pass-check-health
No key found for my-password.gpg
Problems found: 1
```

### 2. ram-by-proc

Example:
```
$ ram-by-proc firefox-bin
4.4G
```
